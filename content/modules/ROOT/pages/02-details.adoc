== Your environment

Everything you need is already deployed and running.
You will not install anything.

[cols="1,3",options="header"]
|===
| Component | What it does

| *OpenShift cluster*
| Base platform, provided via Field Source

| *Red Hat OpenShift AI (RHOAI)*
| Workbench notebook, AI Pipelines (Kubeflow Pipelines v2), model serving integration

| *AI Ops Assistant*
| The four-agent LangGraph application you will observe -- live traffic already flowing

| *Load generator*
| Submits synthetic queries every ~10 seconds -- dashboards are pre-populated when you arrive

| *MLflow*
| Distributed tracing and Prompt Registry -- tracking URI pre-configured as an environment variable in the workbench

| *LokiStack*
| Log aggregation for the `{project_namespace}` namespace

| *Grafana*
| Pre-built AI observability dashboard connected to Prometheus and Loki

| *ChromaDB*
| In-cluster vector store with two collections: RHOAI/OCP docs and SRE runbooks

| *MaaS (Model as a Service)*
| Shared external inference endpoint, OpenAI-compatible API -- no GPU required, no local model deployment

| *AI Pipelines*
| Kubeflow Pipelines v2 enabled on the RHOAI instance
|===

You access everything through the RHOAI workbench and the OpenShift and Grafana web consoles.
No local tooling installation is required.
